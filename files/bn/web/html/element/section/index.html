সারাংশ

HTML সেকশন এলিমেন্ট (<section>) একটি ডকুমেন্ট এর কিছু অংশকে নির্দিষ্ট করে। যেমনঃ শিরোনাম সহ কিছু কন্টেন্ট।

ব্যবহার নির্দেশিকাঃ

  • যদি একটি {{HTMLElement("section")}} এর কন্টেন্ট আলাদাভাবে চিহ্নিত করার প্রয়োজন হয়, তাহলে {{HTMLElement("article")}} এলিমেন্ট ব্যবহার করতে পারেন।
  • সাধারন কন্টেইনার হিসেবে {{HTMLElement("section")}} এলিমেন্ট ব্যবহার করবেন না; সাধারন কন্টেইনার হিসেবে ব্যবহার করার জন্য{{HTMLElement("div")}} এলিমেন্ট রয়েছে। সহজ কথায়, সেকশন শুধু একটি ডকুমেন্ট এর আউটলাইন তৈরিতে ব্যবহার করা যাবে।
  • কন্টেন্ট ক্যাটাগরিফ্লো কন্টেন্ট, সেকশনিং কন্টেন্ট, পাল্পেবল কন্টেন্ট।
  • অনুমোদিত কন্টেন্ট ফ্লো কন্টেন্ট
  • Tag omission {{no_tag_omission}}
  • অনুমোদিত প্যারেন্ট এলিমেন্ট সমূহ যেকোনো এলিমেন্ট, যা ফ্লো কন্টেন্ট সমর্থন করে। তবে খেয়াল রাখবেন, {{HTMLElement("section")}} এলিমেন্ট কে কখনো {{HTMLElement("address")}} এলিমেন্ট এর ভেতরে ব্যবহার করা যাবে না।
  • DOM ইন্টারফেস {{domxref("HTMLElement")}}

অ্যাট্রিবিউট সমূহ

এই এলিমেন্টের মধ্যে শুধু গ্লোবাল অ্যাট্রিবিউট সমূহ ব্যবহার করা যাবে।
