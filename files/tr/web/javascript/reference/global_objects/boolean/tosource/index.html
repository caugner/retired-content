ToSource () metodu, nesnenin kaynak kodunu temsil eden bir dize döndürür.

Sözdizimi

booleanObj.toSource()
Boolean.toSource()

Return değeri

Nesnenin kaynak kodunu temsil eden bir dize.

Açıklama

toSource metodu aşağıdaki değerleri döndürür:

  • Yerleşik(built-in) {{jsxref ("Boolean")}} nesne için toSource, kaynak kodun mevcut olmadığını belirten aşağıdaki dizeyi döndürür: 
    function Boolean() {
    [native code]
}
  • {{Jsxref ("Boolean")}} örneği için toSource, kaynak kodunu temsil eden bir dize döndürür.

This method is usually called internally by JavaScript and not explicitly in code.

Özellikler

Herhangi bir standardın parçası değil. JavaScript 1.3'te uygulandı. 1.3.
