Matematiksel İşaretleme Dili (MathML), matematiksel gösterimi tanımlamak ve hem yapısını ve hemde içeriğini yakalamak için XML'in bir lehçesidir.

Bu güçlü teknolojiyle çalışmanıza yardımcı olacak belgelere, örneklere ve araçlara bağlantılar bulacaksınız. Hızlı bir genel bakış için, Mozilla Zirvesi 2013'teki yenilik fuarlarına ait slaytlara bakın.

MathML Referans

MathML Element Referansı
Masaüstü ve mobil tarayıcılar için her bir MathML öğesi ve uyumluluk bilgileri hakkında detaylar.
MathML Özellik Referansı
Elementlerin görünümünü veya davranışını değiştiren MathML özellikleri hakkında bilgi.
MathML Örnekler
MathML örnekleri ve nasıl çalıştığını anlamanıza yardımcı olacak örnekler.
MathML Yazmak
Önerilen MathML editörlerini ve çıktılarını Web içeriğine nasıl entegre edebileceklerini de içeren MathML yazma önerileri ve ipuçları.

Araçlar

  • W3C Validator
  • FireMath Firefox add-on
  • Mathzilla Firefox add-on koleksiyonu
  • TeXZilla — Javascript LaTeX to MathML dönüştürücüsü (canlı demo, Firefox OS webapp, Firefox add-on, Web Sayfası kullanma, JS program v.b.)
  • LaTeXML - LaTeX dökümanını HTML+MathML Web sayfasına dönüştür
  • Web Denklemleri - Elle yazılmış denklemleri MathML yada LaTeX dönüştür
  • MathJax - Matematik için çapraz tarayıcı JavaScript ekran motoru. MathJax'ı MathML kullanmaya zorlamak için, Mozilla add-on'nunu, Safari uzantısını yada GreaseMonkey scriptini deneyin.
