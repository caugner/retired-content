Sebep: CORS isteği HTTP değil

Yanlış giden ne?

{{Glossary("CORS")}} istekleri yalnızca HTTPS URL şemasını kullanabilir, ancak istek tarafından belirtilen URL farklı türdedir. Bu genellikle URL file:/// kullanarak yerel bir dosya belirtirse olur.

Bu sorunu çözmek için, CORS içeren isteklerde bulunurken HTTPS URL'leri kullandığınızdan emin olun.
