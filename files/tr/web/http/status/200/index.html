HTTP cevap durum kodu 200 OK  isteğin başarılı olduğunu ifade eder. Bu durum koduna sahip cevaplar, varsayılan olarak önbelleğe (cache) alınabilir.

Başarılı isteğin ne anlama geldiği, yapılan HTTP isteğin türüne bağlıdır:

  • {{HTTPMethod("GET")}}: İstenilen kaynak bulundu ve dönen cevabın gövdesinde gönderildi.
  • {{HTTPMethod("HEAD")}}: Bütün başlık mesajları (HEADER) cevabın gövdesi olarak döner.
  • {{HTTPMethod("POST")}}: Yapılan isteğe karşılık gelen sonuç, cevabın gövdesinde gönderilir.
  • {{HTTPMethod("TRACE")}}: Sunucu, aldığı isteği cevabın gövdesi olarak gönderir.

Başarılı olan {{HTTPMethod("PUT")}} veya {{HTTPMethod("DELETE")}} isteği sonucunda dönen cevap genellikle 200 OK değil, {{HTTPStatus("204")}}No Content (veya bir kaynak ilk kez oluşturulmuşsa {{HTTPStatus("201")}}) Created) olur.
