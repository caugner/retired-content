Afecku tamellit yettak-awend tamuɣli ɣef tasedmirt  tamatut,n JavaScript ed timellal n usali asebtar.ad snulfuḍ asekles,naɣ amaɣnu,n wasmel innek deg yiwet n talit n wakud.afecku yagi yeskanay-ak-id tamuɣli tamatut n wayen id yefɣen seg iminig ɣer samaɣnu innek ed tudlift n usnagar n tugna ɣer umaɣnu.

Ad ḥelim kṛad n adahil-ifecka iw sekyed n uger deg lulleḍ n timeẓra n umaɣnu:

  • The Waterfall yeskanay-ed timhalin timeẓliyin yettwagan sɣuṛ iminig,am uselkem n adahil n usebtar, a JavaScript,aten yejgger d uleqwaḍ n iḍuman
  • Aseklu n usiwel yeskanay tiseɣnin n JavaScript ideg iminig yesɛada amur meqṛan n wakud
  • Tafelwit Flame ibeqqeḍ tasemdit usiwel JavaScript di tiddin usekles
  • Iḥeṛṛiyen n iẓṛi ibeqqeḍ iḥeṛṛiyen i tgiḍ s tangalt deg tiddin usekles.timeẓṛiwt agi tettbaned ḥaca ma trecmeḍ «Assayla n usekles»  deg iɣewwaren n ufecku tamellit
