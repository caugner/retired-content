Sommario

L'Elemento HTML Strikethrough  (<strike>) disegna il testo con uno "strikethrough", cioè una linea a metà altezza.

Nota d'uso: Come tutti gli elementi di pura presentazione, {{ HTMLElement("strike") }} è deprecato in HTML 4 e XHTML 1 ed è obsoleto in HTML5. Se semanticamente appropriato, ad es. se rappresenta contenuto cancellato, si usi al suo posto l'elemento {{ HTMLElement("del") }} ; in tutti gli altri casi, si usi un elemento  {{ HTMLElement("span") }} con l'attributo style contenente la proprietà  CSS {{ cssxref("text-decoration") }} impostata a line-through.

Attributi

Questo elemento non ha altri attributi oltre agli attributi globali, comuni a tutti gli elementi.

Interfaccia DOM

Questo elemento implementa l'interfaccia HTMLElement.

Nota di implementazione: fino alla Gecko 1.9.2 inclusa, Firefox implementa per questo elemento l'interfaccia  {{domxref('HTMLSpanElement')}} .

Esempio

<strike>Today's Special: Salmon</strike> NO LONGER AVAILABLE<br />
<span style="text-decoration:line-through;">Today's Special: Salmon</span> SOLD OUT

Risultato

Today's Special: Salmon NO LONGER AVAILABLE
Today's Special: Salmon SOLD OUT

Vedere anche

  • L'elemento {{ HTMLElement("s") }} , alter ego dell'elemento {{ HTMLElement("strike") }} e, come questo, obsoleto.
  • L'elemento {{ HTMLElement("del") }} da utilizzare in sua vece se il dato è stato cancellato.
  • La proprietà CSS {{ cssxref("text-decoration") }} da utilizzare per ottenere il medesimo risultato in visualizzazione ottenibile con l'elemento {{ HTMLElement("strike") }} .
