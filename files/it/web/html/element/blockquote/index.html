Sommario

L'elemento HTML <blockquote> (o Block Quotation) indica che il testo contenuto è una citazione. Solitamente è visualizzato con un'indentazione maggiore del resto del testo. Si può fornire l'URL dal quale è stato citato il testo usando l'attributo {{htmlattrxref("blockquote", "cite")}}, o usando l'elemento {{HTMLElement("cite")}}.

  • Categoria Contenuto di flusso, sezioni, contenuto palpabile.
  • Contenuto permesso Contenuto di flusso.
  • Omissione dei tag {{no_tag_omission}}
  • Elementi genitore permessi Qualsiasi elemento che accetti contenuto di flusso.
  • Interfaccia DOM {{domxref("HTMLQuoteElement")}}

Attributi

Questo elemento include gli attributi globali.

{{htmlattrdef("cite")}}
L'URL del documento dal quale è stato informato il testo o un messaggio riguardante le informazioni citate.

Esempio

<blockquote cite="http://developer.mozilla.org">
  <p>Questa è una citazione presa da MDN.</p>
</blockquote>

Risultato

Questa è una citazione presa da MDN.
