Sommario

L'elemento HTML <br> (o Line Break) crea un "a capo" nel testo. È utile quando è importante la divisione tra le linee, ad esempio in una poesia.

Non usare l'elemento <br> per aumentare lo spazio tra due linee di testo, ma le proprietà CSS {{cssxref("margin")}} o {{cssxref("line-height")}} applicate all'elemento {{HTMLElement("p")}}.

  • Categoria Contenuto di flusso, contenuto testuale.
  • Contenuto permesso Nessuno, è un {{Glossary("elemento vuoto")}}.
  • Omissione dei tagDeve esserci il tag di apertura ma non ci può essere il tag di chiusura. Nei documenti XHTML, questo elemento va scritto come <br />.
  • Elementi genitore permessiQualsiasi elemento che accetta contenuto testuale.
  • Interfaccia DOM {{domxref("HTMLBRElement")}}

Attributi

Questo elemento include gli attributi globali.

{{htmlattrdef("clear")}} {{deprecatedGeneric('inline','HTML4.01')}} {{obsoleteGeneric('inline','HTML5')}}
Con questo attributo si può ignorare l'allineamento della linea precedente. Può assumee i valorei left, right e all.

Nota: Questo attributo è obsoleto in HTML5 e non dovrebbe essere usato: gli sviluppatori dovrebbero usare la proprietà CSS {{cssxref("clear")}}.

Esempio

Mozilla Foundation<br>
1981 Landings Drive<br>
Building K<br>
Mountain View, CA 94043-0801<br>
USA
