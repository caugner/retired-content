Sommario

L'elementho HTML <big> ingrandisce il testo di un livello (ad esempio, da piccolo a medio, o da grande a molto grande), fino alla dimensione massima permessa dal browser.

Nota: Siccome questo elemento è solo stilistico, è stato rimosso in HTML5 e non dovrebbe essere usato. Gli sviluppatori dovrebbero utilizzare la proprietà CSS {{cssxref("font-size")}}.

Attributi

Questo elemento supporta solo gli attributi globali, comuni a tutti gli elementi.

Esempio

<p>
  Questa è la prima frase. <big>Questa è scritta più grande.</big>
</p>

Lo stesso effetto può essere ottenuto usando il CSS:

<p>
  Questa è la prima frase. <span style="font-size:1.2em">Questa è scritta più grande.</span>
</p>

Risultato

Questa è la prima frase. Questa è scritta più grande.

Interfaccia DOM

Questo elemento implementa l'interfaccia {{domxref("HTMLElement")}}.

Nota: Fino a Gecko 1.9.2, Firefox implementava l'interfaccia {{domxref("HTMLSpanElement")}}.
