Sommario

L'elemento HTML nav (<nav>) rappresenta una sezione della pagina che contiene link ad altre pagine o a parti della pagina corrente: una sezione di navigazione.

Note d'uso:

  • Non tutti i link di un documento devono essere in un elemento <nav>, che è stato pensato solo per i blocchi di link più importanti; spesso l'elemento {{HTMLElement("footer")}} contiene una lista di link che non necessitano di essere in un elemento {{HTMLElement("nav")}}.
  • Un documento potrebbe avere vari elementi {{HTMLElement("nav")}}, per sempio uno per la navigazione all'interno del sito e uno per la navigazione all'interno della pagina.
  • Gli interpreti, come un programma di lettura dello schermo per gli utenti disabili, può usare quest'elemento per determinare se non visualizzarne il contenuto.
  • Categorie di contenuti Contenuti di flusso, contenuti di sezionamento, contenuti palpabili.
  • Contenuti permessi Contenuti di flusso.
  • Omissione del tag {{no_tag_omission}}
  • Permitted parent elements Qualsiasi elemento che accetti contenuti di flusso.
  • DOM interface {{domxref("HTMLElement")}}

Attributi

Questo elemento include solo gli attributi globali.

Esempi

<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Contatti</a></li>
  </ul>
</nav>
