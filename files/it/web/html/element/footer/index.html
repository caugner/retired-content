Sommario

L'elemento HTML <footer> rappresenta il footer della sua sezione più vicina. Un footer solitamente contiene informazioni sull'autore della sezione, il copyright o link a documenti correlati.

Note d'uso:

  • Racchiudere le informazioni sull'autore in un elemento {{HTMLElement("address")}}, che può essere posizionato nell'elemento <footer>
  • L'elemento <footer> non rappresenta una sezione del contenuto e quindi non introduce una nuova sezione nell'indice.
  • Categorie di contenuto Contenuti di flusso, contenuti palpabili.
  • Contenuti permessi Contenuti di flusso, esclusi <footer> e {{HTMLElement("header")}}.
  • Omissione del tag {{no_tag_omission}}
  • Elementi genitore permessi Qualsiasi elemento che accett contenuti di flusso. Notare che un elemento <footer> non può essere discendente di un elemento {{HTMLElement("address")}}, {{HTMLElement("header")}} o di un altro <footer>.
  • Interfaccia DOM {{domxref("HTMLElement")}}

Attributi

Questo elemento include solo gli attributi globali.

Esempi

<footer>
  Il copyright o delle informazioni su di un articolo
</footer>
