{{Non-standard_header}}{{CSSRef}}

Sommario

La pseudo-classe :-moz-list-bullet è usata per modificare il pallino degli elementi lista.

Sintassi

li::-moz-list-bullet {proprietà di stile }

Esempi

.liste li::-moz-list-bullet{ font-size:36px }

...applicato a...

<ul>
<li>Number 1</li>
<li>Number 2</li>
<li>Number 3</li>
</ul>
<ul class="liste">
<li>Number 1</li>
<li>Number 2</li>
<li>Number 3</li>
</ul>

...risultato..
