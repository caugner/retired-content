Sommario

Riporta la finestra corrente all'ultima pagina visitata, o alla pagina più indietro nella Cronologia del browser.

Sintassi

window.back()

Esempio

function vaiIndietro() {
   if ( possoAndareIndietro )
      window.back();
}

Specifiche

DOM Level 0. Non è parte di alcuno standard.
