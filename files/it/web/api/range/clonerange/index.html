Restituisce un oggetto Range con i punti d'inizio e di fine identici a quelli del Range attuale.

Sintassi

clone = range.cloneRange();

Esempio

range = document.createRange();
range.selectNode(document.getElementsByTagName("div").item(0));
clone = range.cloneRange();

Note

Il Range viene copiato per valore, non per referenza, quindi una modifica su uno dei due oggetti non avr√† effetti sull'altro.
