Questa pagina contiene un elenco di modifiche in Firefox 1.5, che coinvolgono gli sviluppatori XUL.

Cambiamenti specifici

  • Modifica dell'API tree
  • Caratteri internazionali nei JavaScript XUL (ha efficacia solo su quelle estensioni che hanno dei file JavaScript contenenti caratteri non-ASCII)
  • Modiche di XMLHttpRequest
  • Cambiamenti di XUL per Firefox 1.5
  • XPCNativeWrapper sono attivati come opzione predefinita, e il comportamento è in parte cambiato rispetto alla versione 1.0.x
  • Un metodo più semplice di Registrazione Chrome rende obsoleto contents.rdf.
  • Per i menu contestuali con overlay: la funzione gContextMenu.linkURL() è stata rinominata come gContextMenu.getLinkURL(), e linkURL ora è una proprietà. Per utilizzarla mantenendo la retrocompatibilità:
    url = 'getLinkURL' in gContextMenu ? gContextMenu.getLinkURL() : gContextMenu.linkURL();
