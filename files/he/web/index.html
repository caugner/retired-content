האופי הפתוח של הרשת העולמית חושף אפשרויות מדהימות לאנשים שרוצים ליצור אתרים או אפליקציות מקוונות. כדי לנצל את יכולות הרשת באופן מרבי, עליך לדעת כיצד להשתמש בהם. חקור את הקישורים למטה כדי ללמוד עוד אודות טכנולוגיות רשת שונות.

טכנולוגיות לפיתוח אינטרנטי

מידע בסיסי

HTML
שפת סימני עריכה לתמליל-על (HyperText Markup Language) הינה שפה המשמשת לתאר וליצור תוכן של דפי אינטרנט ומסמכים אחרים הניתנים לצפייה בדפדפן אינטרנט.
CSS
גיליונות סגנון מדורגים (Cascading Style Sheets) משמשים לתאר את מראה או תצוגת התוכן בדף האינטרנט.
HTTP
פרוטוקול תעבורת תמליל-על (HyperText Transfer Protocol) משמש להעביר HTML ומסמכי היפר-מדיה ברשת.

סקריפטים

JavaScript
JavaScript הינה שפת תכנות שרצה בדפדפן שלך. ניתן להשתמש בה להוספת אינטראקטיביות ודינמיות לאתר או אפליקציה שלך.
לאור כניסתו של Node.js, ניתן להריץ על השרת.
Web APIs
ממשקי תכנות אפליקציות רשת (Web Application Programming Interfaces) משמשים לבצע משימות, כמו שינוי ה- DOM, הפעלת שמע או וידאו,או יצירת גרפיקה בתלת-מימד.
  • אינדקס ממשקי ה- API ברשת מפרט את טיפוסי האובייקטים בהם ניתן להשתמש בפיתוח בסביבת הרשת.
  • עמוד ה- WebAPI מפרט ממשקי תקשורת, חומרה ואחרים בהם ניתן להשתמש באפליקציות רשת.
  • אינדקס ה- Events מפרט את האירועים בהם ניתן להשתמש כדי לעקוב ולהגיב לדברים מעניינים שקרו בעמוד או באפליקציה.
Web Components (רכיבי Web)
רכיבי Web הינם אוסף טכנולוגיות המאפשר ליצור אלמנטים בהתאמה אישית הניתנים לשימוש חוזר — כאשר הפונקציונליות שלהם מבודדת משאר הקוד — ולהשמיש אותם באפליקציות הרשת שלך.

גרפיקה

קנבס (Canvas)
אלמנט {{HTMLElement("canvas")}} מספק API ליצירת גרפיקות תלת-מימד באמצעות שימוש ב- TextScript.
SVG
Scalable Vector Graphics (גרפיקה וקטורית ברת-מידות SVG) מאפשרת להשתמש בקווים, עקובומות וצורות גאומטריות אחרות ליצירת גרפיקה. באמצעות וקטורים, ניתן ליצור תמונות שמותאמות באופן נקי לכל גודל.
WebGL
WebGL הינו JavaScript API המאפשר לצייר גרפיקות דו-מימד או תלת-מימד, באמצעות אלמנט {{HTMLElement("canvas")}}. טכנולוגיה זו מאפשר להשתמש ב- OpenGL ES בתשתית רשת.

שמע ומולטימדיה

טכנולוגיות מדיה ברשת
רשימה של API הקשורים למדיה, עם קישורים לתיעוד הדרוש לכל אחד.
סקירה של טכנולוגית מדיה ברשת
מבט כללי על טכנולוגיות רשת פתוחות ו- API המספקים תמיכה לשמע ווידאו. אם אינך בטוח באיזה API להשתמש, זה המקום להתחיל ממנו.
API ללכידת מדיה וזרמים (streams)
אינדקס ל- API שמאפשר להזרים (סטרימינג), להקליט ולתמרן מדיה באופן מקומי וברשת. כולל שימוש במצלמות ומיקרופונים מקומיים ללכידת וידאו, שמע וסטילס.
שימוש בשמע ווידאו עם HTML
הטמעת וידאו ו/או שמע בעמוד רשת ושליטה בפלייבק שלו.
WebRTC
RTC הוא קיצור ל- Real-Time Communications (תקשורת בזמן-אמת), טכנולוגיה שמאפשרת להזרים שמע / וידאו ולשתף מידע בין משתמשי דפדפנים (עמיתים - peers).

אחר

MathML
Mathematical Markup Language (שפת סימני עריכה מתמטית - MathML) מאפשרת להציג משוואות ותחביר מתמטיות.
XSLT
Extensible Stylesheet Language Transformations (טרנספורמציית שפת עיצוב ברת-הרחבה - XSLT) מאפשרת להמיר מסמכי XML לתוכן HTML קריא יותר.
EXSLT
פונקציות נוספות המספקות אפשרויות נוספות ל- XSLT.
XPath
XPath מאפשר לבחור צמתים במסמך, על-ידי שימוש בתחביר חזק יותר מזה של CSS selectors.

אזור הלמידה

לימוד פיתוח ברשת
אוסף מאמרים המספק למתחילים כל מה שצריך כדי להתחיל לתכנת אתרים פשוטים.
אפליקציות רשת מתקדמות (Progressive web apps)
אפליקציות רשת מתקדמות (PWA - Progressive Web Apps) משתמשות ב- Web API מודרניים ביחד עם אסטרטגיית שיפור מתקדמת ליצירת אפליקציות רשת מרובות-פלטפורמות. אפליקציות אלו עובדות בכל מקום, ומספקות מספר אפשרויות עם אותם יתרונות של חווית משתמש כמו אפליקציות native. אוסף מסמכים ומדריכים זה יראו את כל מה שיש לדעת לגבי PWA.

נושאים נוספים

פיתוח אפליקציות רשת
אוסף מאמרים זה מסביר את הטכניקות המשמשות בפיתוח אפליקציות רשת לסביבת מכשירים ניידים (מובייל) ונייחים (מחשבים).
נגישות
אתרים נגישים מאפשרים לאנשים רבים ככל האפשר להשתמש ברשת, לרבות בעל מוגבלויות ראייה, שמיעה ואחרות. אוסף מאמרים זה מספק מידע לגבי פיתוח רשת נגיש.
ביצועים ברשת
ביצועים ברשת הינה אמנות הווידוא שאפליקציות רשת ייטענו מהר ויגיבו לפעולות משתמש, בלי קשר לרוחב הפס, גודל המסך, רשת, או יכולות המכשיר.
לוקליזציה (L10n) וגלובליזציה / אינטרנשיונליזציה (I18n)
הרשת היא קהילה גלובלית. חשוב לוודא שהאתר שלך יהיה חלק ממנה, באמצעות הקפדה על מתן תוכן האתר בשפה ופריסה המצופים על-ידי משתמשי היעד שלך.
אבטחה
אל תיתן לאתר או לאפליקציה שלך להדליף מידע פרטי לחבר'ה הרעים. תוכל להשתמש באוסף מאמרים זה כדי לוודא שהפרוייקטים שלך מאובטחים.
WebAssembly
WebAssembly הינה שפה חדשה שניתן להריץ בדפדפנים חדישים — זוהי שפת-סף דמוית-אסמפלי עם פורמט בינארי קומפקטי בעל ביצועים כמעט ברמת native, ומספק לשפות כמו C / C++ קומפילצית-יעד, כך שהן יכולות לרוץ ברשת.
