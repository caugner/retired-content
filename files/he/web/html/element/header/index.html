תגית <header> מייצגת את החלק העליון של מסמך ה-HTML. 
אזור זה משמש ככותרת עבור <section> או בעבור כל הדף.
מלבד כותרת אזור זה מכיל תוכן עניינים, תיבת חיפוש, לוגו של חברה ועוד.
