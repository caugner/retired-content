תגית <footer> מייצגת את התפריט / האזור התחתון במסמך ה-HTML. תחת תגית זו נהוג לכתוב זכויות יוצרים, קישורים, תנאי שימוש, טופס צור קשר, הרשמה לעדכונים ועוד.
