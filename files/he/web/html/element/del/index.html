תארים

תגית זו נכללת ברשימת התארים הגלובליים.

  • {{htmlattrdef("cite")}} - תכונה המסבירה את מהות השינוי.
  • {{htmlattrdef("datetime")}} - תכונה זו מציינת את השעה והתאריך של השינוי וחייבת להיות מחרוזת תאריך חוקית.
