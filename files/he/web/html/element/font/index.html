תגית <font> מגדירה את שם הפונט, צבע וגודל הטקסט במסמך ה-HTML.

* זוהי תגית ישנה ואינה חלק משפת HTML גרסה 5, אלמנט זה עשוי לעבוד בכמה דפדפנים אך יתכן שלא יתמך בעתיד. יש להימנע משימוש בתגית זו.

תארים

תגית זו נכללת ברשימת התארים הגלובליים.

  • {{htmlattrdef("color")}} - תכונה זו קובעת את צבע הטקסט באמצעות שם הצבע או בפורמט הקסדצימלי #RRGGBB.
  • {{htmlattrdef("face")}} - תכונה זו קובעת את הגופן בו הטקסט יוצג. 
  • {{htmlattrdef("size")}} - תכונה זו מציינת את גודל הגופן כערך מספרי או יחסי. הערכים המספריים נעים בין 1 ל-7 כאשר 1 הוא הקטן ביותר ו-3 ברירת המחדל.

דוגמא

<body>
<font size="5" color="red">Please Write Your Text Here!</font>
</body>
