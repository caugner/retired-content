תגית <aside> מייצגת את התפריט / האזור הצדדי במסמך ה-HTML. תחת תגית זו נהוג להוסיף פרסומות, ציטוטים מטקסטים, תוספים, ורכיבי תוכן (ווידג'טים) המציגים מידע בצורה נגישה וישירה.
