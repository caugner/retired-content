תארים

תגית זו נכללת ברשימת התארים הגלובליים.

href - משמש להוספת כתובת קישור.
target - קובע היכן יפתח הקישור (עמוד הנוכחי, כרטיסייה חדשה וכו').
id - מזהה המשמש לקישור פנימי בין עמודים.
