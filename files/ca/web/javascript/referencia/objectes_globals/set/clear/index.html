{{JSRef}}

El mètode clear() esborra tots els elements emmagatzemats dins l'objecte Set.

Sintaxi

mySet.clear();

Exemples

Utilitzar el mètode clear

var mySet = new Set();
mySet.add(1);
mySet.add("foo");

mySet.size;       // 2
mySet.has("foo"); // true

mySet.clear();

mySet.size;       // 0
mySet.has("bar")  // false
