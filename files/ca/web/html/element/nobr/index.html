Sumari

L'element HTML <nobr> impedeix que un text es trenqui automàticament en una nova línia, per la qual cosa es mostra en una línia llarga i pot ser necessari desplaçar-se. Aquesta etiqueta no és HTML estàndard i no ha d'utilitzar-se. En el seu lloc, utilitzi la propietat CSS {{Cssxref("white-space")}} de la següent manera:

<span style="white-space: nowrap">Long line with no breaks</span>
