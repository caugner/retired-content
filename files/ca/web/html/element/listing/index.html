Sumari

L'Element HTML Llistat (<listing>) processa el text entre les etiquetes d'inici i fi sense interpretar l'HTML entre elles i usar una font monospaced. L'estàndard HTML 2 recomana que les línies no es trenquin quan no superin els 132 caràcters.

Nota: No utilitzeu aquest element
  • Està obsolet des d'HTML 3.2 i no ha estat implementat per tots els navegadors, ni de manera coherent. Més encara, està obsolet en HTML5 i pot ser representat pels agents d'usuari conformes com l'element {{HTMLElement("pre")}}, que interpretarà l'HTML intern!
  • En el seu lloc, utilitzi l'element {{HTMLElement("pre")}} o si s'adapta semànticament l'element {{HTMLElement("code")}}, eventualment escapa de l'HTML '<'and'>' perquè no s'interpreti.
  • Una font monoespaciada també es pot obtenir en un element simple {{HTMLElement("div")}}, aplicant un estil CSS adequat usant monospace com el valor de la font genèrica en una propietat {{cssxref("font-family")}}.

Atributs

Aquest element no té altres atributs que els atributs globals, comuns a tots els elements.

Interfície DOM

Aquest element implementa la interfície {{domxref('HTMLElement')}}.

Nota d'implementació: fins Gecko 1.9.2 inclosos, Firefox implementa la interfície {{domxref('HTMLSpanElement')}} per aquest element.

Veure

  • Els elements {{HTMLElement("pre")}} i {{HTMLElement("code")}} per a ser utilitzat al seu lloc.
  • Els elements {{HTMLElement("plaintext")}} i {{HTMLElement("xmp")}}, similars a {{HTMLElement("listing")}} també estan obsolet.
