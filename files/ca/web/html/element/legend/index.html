L'Element HTML <legend> (o HTML llegenda element de camp) representa un títol per al contingut del se element principal  {{HTMLElement("fieldset")}}.

Categories de contingut Cap.
Contingut permès Phrasing content.
Omissió de l'etiqueta {{no_tag_omission}}
Elements pares permesos En un {{HTMLElement ("fieldset")}} el primer fill és aquest element <legend>
Interfície DOM {{domxref("HTMLLegendElement")}}

Atributs

Aquest element nomès inclou els atributs globals.
