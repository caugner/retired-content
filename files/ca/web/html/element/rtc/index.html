Sumari

L'Element HTML <rtc> abasta notacions semàntiques dels caràcters presentats en un ruby d'elements {{HTMLElement("rb")}} utilitzats a l'interior de l'element {{HTMLElement ("ruby")}}. Els elements {{HTMLElement ("rb")}} poden tenir tant la pronunciació ({{HTMLElement("rt")}}) com notacions semàntiques ({{HTMLElement("rtc")}}) .

Categories de contingut Cap.
Contingut permès Phrasing content or elements {{HTMLElement("rt")}} .
Omissió de l'etiqueta L'etiqueta de tancament es pot ometre si és seguit immediatament per una etiqueta d'obertura de l'element {{HTMLElement("rb")}}, {{HTMLElement("rtc")}} o {{HTMLElement("rt")}} o per la seva etiqueta de tancament dels pares.
Elements pares permesos Un element {{HTMLElement("ruby")}}.
Interfície DOM {{domxref("HTMLElement")}}

Atributs

Aquest element només inclou els atributs globals.

Exemple

<ruby>
   <rb>旧</rb>
   <rb>金</rb>
   <rb>山</rb>
   <rt>jiù</rt>
   <rt>jīn</rt>
   <rt>shān</rt>
   <rtc>San Francisco</rtc>
</ruby>
