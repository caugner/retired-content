Sumari

L'element HTML directori  (<dir>) representa un directori, és a dir, una col·lecció de noms d'arxiu.

Nota d'ùs: No utilitzeu aquest element. Encara present en l'especificació HTML inicial, s'ha desaprovat en HTML 4, llavors és obsoleta en HTML5. Utilitzar el {{HTMLElement("ul")}} al seu lloc

Interfície DOM

Aquest element implementa la interfície {{domxref("HTMLDirectoryElement")}}.

Atributs

Igual que tots els altres elements HTML, aquest element és compatible amb els atributs globals.

{{htmlattrdef("compact")}}
Aquest atribut booleà aconsella que la llista hauria de traduir-se en un estil compacte. La interpretació d'aquest atribut depèn de l'agent d'usuari i no funciona en tots els navegadors.
Nota d'ùs: No utilitzeu aquest atribut, ja que ha quedat desprovat: l'element {{HTMLElement("dir")}} s'ha de donar estil amb CSS. Per donar un efecte similar que l'atribut compact, la propietat CSS {{cssxref("line-height")}} es pot utilitzar amb un valor del 80%.

Veure

  • Altres elements HTML relacionats: {{HTMLElement("ol")}}, {{HTMLElement("ul")}}, {{HTMLElement("li")}}, {{HTMLElement("menu")}};
  • Propietats CSS que poden ser especialment útils per estilitzar l'element <dir>:
    • La propietat {{cssxref('list-style')}}, es útil per triar la forma en què es mostra l'ordinal.
    • CSS counters, es útil per gestionar llistes niades complexes.
    • La propietat {{Cssxref('line-height')}}, es útil per simular el desús de l'atribut {{htmlattrxref("compact", "dir")}}.
    • La propietat {{cssxref('margin')}}, es útil per controlar la sagnia de la llista.
