Sumari

L'element HTML <image> era un element experimental dissenyat per mostrar les imatges. Mai es va aplicar i la norma es la utilització del element {{HTMLElement("img")}}.

No utilitzeu aquest! Per tal de mostrar les imatges, utilitzeu l'estàndard de l'element {{HTMLElement("img")}}.

Recordeu que fins Firefox 22, encara que no suportat, un element <image> es va associar amb un {{domxref("HTMLSpanElement")}}. Després es va corregir i ara s'associa amb un {{domxref("HTMLElement")}} per requeriment de l'especificació.
