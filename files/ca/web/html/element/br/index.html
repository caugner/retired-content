Resum

L'element HTML <br>  (de l'Anglès Line Break) produeix un salt de línia en el text (carriage-return). És útil per escriure un poema o una adreça, on la divisió de línies és significatiu.

No s'ha de fer servir <br> per augmentar la distància entre línies de text; sinó la propietat CSS {{cssxref('margin')}} o l'element {{HTMLElement("p")}} .

  • Categories de contingut Contingut de flux, phrasing content.
  • Contingut permès Cap, és un  {{Glossary("empty element")}}.
  • Omissió de l'etiqueta Ha de tenir una etiqueta d'entrada, i cap de sortida. En els documents XHTML, s'ha d'escriure aquest element com a <br />.
  • Elements pares permesos Qualsevol element que accepti phrasing content.
  • Interfície DOM {{domxref("HTMLBRElement")}}

Atributs

Aquest element inclou els atributs globals.

{{htmlattrdef("clear")}} {{deprecatedGeneric('inline','HTML4.01')}} {{obsoleteGeneric('inline','HTML5')}}
Indica on comença la següent línia després del salt de línia .

Nota d'ús: Aquest atribut és obsolet en {{HTMLVersionInline(5)}} i no s'hauria de fer servir per desenvolupadors. Es pot fer servir la propietat CSS {{CSSxref('clear')}}  per això.

Exemple

Mozilla Foundation<br>
1981 Landings Drive<br>
Building K<br>
Mountain View, CA 94043-0801<br>
USA

L'HTML de dalt mostra:

Mozilla Foundation
1981 Landings Drive
Building K
Mountain View, CA 94043-0801
USA
