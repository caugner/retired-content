Είμαστε μια ανοιχτή κοινότητα προγραμματιστών και συγγραφέων που δημιουργούμε πόρους για ένα καλύτερο διαδίκτυο, ανεξαρτήτως μάρκας, προγράμματος περιήγησης, ή πλατφόρμας. Οποιοσδήποτε μπορεί να συνεισφέρει και να μάς κάνει δυνατότερους. Μαζί, μπορούμε να συνεχίσουμε να οδηγούμε την καινοτομία στο διαδίκτυο για να υπηρετήσουμε το γενικότερο καλό. Όλα ξεκινούν εδώ, μαζί σας.

Κάθε μέρος του MDN (από τον κώδικα που καθιστά την ιστοσελίδα λειτουργική, έως και την τεκμηρίωση, τα δείγματα κώδικα και τα έτοιμα παραδείγματα) έχει δημιουργηθεί από μια κοινότητα προγραμματιστών και συγγραφέων. Ο καθένας έχει κάτι να προσφέρει και ελπίζουμε ότι θα ενωθείτε μαζί μας!

3 απλά βήματα για το MDN

Το MDN είναι ένα wiki, όπου ο καθένας μπορεί να προσθέσει και να επεξεργαστεί το περιεχόμενο. Δεν χρειάζεται να είστε προγραμματιστής ή να γνωρίζετε πολλά για την τεχνολογία. Υπάρχουν πολλά πράγματα που πρέπει να γίνουν, από απλές εργασίες, όπως η επιθεώρηση κειμένων και η διόρθωση λαθών, έως και πολύπλοκες, όπως η σύνταξη των εγγράφων των API. Αυτός ο οδηγός θα σάς βοηθήσει να ξεκινήσετε και θα σάς κατευθύνει έτσι, ώστε να βρείτε τρόπους για να βελτιώσετε κι εσείς το περιεχόμενο του MDN.

Η συνεισφορά είναι εύκολη και ασφαλής. Ακόμα και αν κάνετε κάποιο λάθος, αυτό μπορεί να διορθωθεί εύκολα· αν δεν ξέρετε ακριβώς πώς πρέπει να είναι τα πράγματα, ή η γραμματική σας δεν είναι και τόσο καλή, μην ανησυχείτε γι 'αυτό! Έχουμε μια ομάδα ατόμων, των οποίων η δουλειά είναι να εξασφαλίζουν ότι το περιεχόμενο του MDN είναι όσο το δυνατόν καλύτερο. Κάποιος θα είναι μαζί σας για να βεβαιωθεί ότι η εργασία σας είναι τακτοποιημένη και καλογραμμένη. Μοιραστείτε τις γνώσεις σας, ακολουθήστε τις δυνάμεις σας και εμπιστευτείτε την υπόλοιπη κοινότητα, που θα σάς βοηθήσει να βελτιώσετε ακόμη περισσότερο την εργασία σας.

Βήμα 1: Δημιουργήστε ένα λογαριασμό στο MDN

Για να ξεκινήσετε τη συνεισφορά σας στο MDN, θα πρέπει να έχετε ένα λογαριασμό στο MDN. Για περισσότερες λεπτομέρειες, παρακαλούμε δείτε πώς μπορείτε να δημιουργήσετε ένα λογαριασμό. Σημειώστε ότι θα χρειαστείτε ένα λογαριασμό GitHub πριν μπορέσετε να δημιουργήσετε ένα λογαριασμό MDN, αφού χρησιμοποιούμε το GitHub για πιστοποίηση, προς το παρόν.

Βήμα 2: Επιλέξτε μια εργασία

Τώρα που έχετε συνδεθεί, διαβάστε τις περιγραφές των διαφορετικών τύπων εργασιών στην {{anch("Πιθανοί τύποι εργασιών", "παρακάτω λίστα")}} και αποφασίστε ποιος τύπος σάς ταιριάζει περισσότερο. Μπορείτε να επιλέξετε οποιαδήποτε εργασία επιθυμείτε και να ξεκινήσετε τη συνεισφορά σας.

Αν η εργασία σας συμπεριλαμβάνει τη δημιουργία νέων σελίδων, παρακαλούμε δείτε το {{SectionOnPage("/en-US/docs/MDN/Contribute/Howto/Create_and_edit_pages", "Getting page creation permissions")}} για σημαντικές πληροφορίες σχετικά με το πώς να λάβετε δικαιώματα για την προσθήκη νέων σελίδων· για λόγους ασφαλείας, οι νέοι λογαριασμοί δεν έχουν αυτή την ικανότητα από προεπιλογή.

Βήμα 3: Κάντε την εργασία

Μόλις αποφασίσετε τι είδους εργασία θέλετε να κάνετε, βρείτε μια συγκεκριμένη σελίδα, παράδειγμα κώδικα κ.λπ., για να εργαστείτε και απλά κάντε το!

Μην σάς ανησυχεί η τελειότητα· άλλοι συνεισφέροντες του MDN είναι εδώ για να σάς βοηθήσουν να διορθώσετε τα λάθη που ξεγλίστρησαν. Αν έχετε ερωτήσεις στην πορεία, δείτε την κοινοτική σελίδα για πληροφορίες σχετικά με τις λίστες αλληλογραφίας και τα κανάλια συνομιλιών, όπου μπορείτε να βρείτε απαντήσεις.

Αν θέλετε να πειραματιστείτε με την επεξεργασία του MDN για να κάνετε κάτι "πραγματικό", έχουμε μια σελίδα Sandbox όπου μπορείτε να παίξετε. Παρακαλούμε περιορίστε τα πειράματά σας σε αυτή τη σελίδα. Παρακαλούμε μην κάνετε περιττές αλλαγές στις σελίδες περιεχομένου, μόνο και μόνο για να δείτε τι θα συμβεί, κάτι που δημιουργεί ένα χάος για τους άλλους να καθαρίσουν, ή ακόμη χειρότερα, συγχύζει τους αναγνώστες που προσπαθούν απλά να μάθουν κάτι ή ψάχνουν κάτι.

Όταν τελειώσετε με την επιλεγμένη εργασία σας, μη διστάσετε να επιλέξετε ένα άλλο στοιχείο, ή δείτε παρακάτω τα άλλα πράγματα που μπορείτε να κάνετε στο MDN.

Πιθανοί τύποι εργασιών

Υπάρχουν πολλοί δρόμοι που μπορείτε να ακολουθήσετε για να συνεισφέρετε στο MDN, ανάλογα με το σύνολο των ικανοτήτων και των ενδιαφερόντων σας. Ακόμα κι αν ορισμένες εργασίες ενδέχεται να είναι αποκαρδιωτικές, έχουμε πολλές απλές δραστηριότητες που διατίθενται. Πολλές από αυτές απαιτούν μόνο πέντε λεπτά (ή και λιγότερο!) από το χρόνο σας. Παρακάτω, μαζί με την εργασία και τη σύντομη περιγραφή της, θα βρείτε τον κατά προσέγγιση χρόνο που απαιτεί ο κάθε τύπος εργασίας συνήθως.

Επιλογή 1: Μου αρέσουν οι λέξεις

Μπορείτε να μάς βοηθήσετε να αξιολογήσουμε ή να τροποποιήσουμε τα υπάρχοντα έγγραφα και να εφαρμόσουμε τις σωστές ετικέτες.

  • Ρύθμιση περίληψης για μια σελίδα (5-15 λεπτά)
  • Αξιολογήσεις επεξεργασίας (5-30 λεπτά)
  • Προσθήκη νέας καταχώρησης στο Γλωσσάριο (15-60 λεπτά)
  • Σύνταξη άρθρου για να βοηθήσετε τους συνανθρώπους σας να μάθουν για το διαδίκτυο (1-3 ώρες)
  • Προσθήκη/Αφαίρεση ετικετών από σελίδες (15-60 λεπτά)
Αν αξιολογείτε άρθρα ή συντάσσετε καινούρια, σάς ζητούμε να εξετάσετε τον Οδηγό στυλ. Έτσι, θα διασφαλιστεί η συνοχή των άρθρων.

Επιλογή 2: Μου αρέσει ο κώδικας

Χρειαζόμαστε περισσότερα δείγματα κώδικα! Μπορείτε, επίσης, να βοηθήσετε στην ανάπτυξη της πλατφόρμας της ιστοσελίδα μας, Kuma, ή να βοηθήσετε στον εμπλουτισμό της βάσης δεδομένων μας σχετικά με τα δεδομένα συμβατότητας των προγραμμάτων περιήγησης.

  • Μετατροπή δειγμάτων κώδικα για να είναι "ζωντανά" (30 λεπτά)
  • Διαβάστε τον οδηγό "Συνεισφορά" (30 λεπτά)
  • Δημιουργία ενός περιβάλλοντος ανάπτυξης του Kuma (1 ώρα)
  • Αποστολή διορθώσεων κώδικα στην βάση κώδικα του Kuma (1 ώρα)
  • Προσθήκη ή ενημέρωση δεδομένων συμβατότητας των προγραμμάτων περιήγησης (30 λεπτά)

Επιλογή 3: Μου αρέσουν και οι λέξεις και ο κώδικας

Έχουμε εργασίες που απαιτούν τεχνικές και γλωσσικές δεξιότητες, όπως η σύνταξη νέων άρθρων, η εξέταση της τεχνικής ακρίβειας, ή η προσαρμογή εγγράφων.

  • Προώθηση του MDN στη δική σας ιστοσελίδα (5 λεπτά)
  • Αφαίρεση "πειραματικών" μακροεντολών από στοιχεία που δεν είναι πλέον πειραματικά (5-30 λεπτά)
  • Τεχνικές αξιολογήσεις (30 λεπτά)
  • Σύνταξη νέου άρθρου σχετικά με ένα θέμα που απαιτείται τώρα (1 ώρα ή περισσότερο)
  • Δημιουργία διαδραστικής άσκησης για να βοηθήσετε τους συνανθρώπους σας να μάθουν για το διαδίκτυο (1 ώρα ή περισσότερο)
  • Διόρθωση σφάλματος στην τεκμηρίωση από την κατηγορία MDN στο Ahoy Bugs (1 ώρα ή περισσότερο)

Επιλογή 4: Θέλω το MDN στη γλώσσα μου

Όλες οι εργασίες μεταφράσεων στο MDN γίνονται απο την καταπληκτική κοινότητά των εθελοντών μας.

  • Μετάφραση σελίδων (2 ώρες) (Δείτε τις κορυφαίες προτεραιότητες)
  • Σύνδεση με άλλους μεταφραστές από τα έργα μετάφρασης (30 λεπτά)

Επιλογή 5: Βρήκα κάποιες λανθασμένες πληροφορίες, αλλά δεν ξέρω πώς να τις φτιάξω

Μπορείτε να αναφέρετε προβλήματα καταθέτοντας ένα σφάλμα τεκμηρίωσης. (5 λεπτά)

Άλλα πράγματα που μπορείτε να κάνετε στο MDN

  • Συμμετοχή στην κοινότητα του MDN.
  • Συμπλήρωση του προφίλ σας έτσι, ώστε οι άλλοι να μπορούν να μάθουν περισσότερα για εσάς.
  • Μάθετε περισσότερα σχετικά με τη συνεισφορά στο MDN.
