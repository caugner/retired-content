JavaScript (जावास्क्रिप्ट ) एक हलकी, ऑब्जेक्ट ओरिएंटेड स्क्रिप्टिंग भाषा है। यह ज्यादातर वेब पेज़ेस के लिए इस्तेमाल की जाती है, पर ब्राउज़र के आलावा दूसरे एनवायरनमेंट जैसे Node.js, Apache CouchDB और Adobe Acrobat में भी इस्तेमाल कर सकते हैं। 

साइट का यह भाग सिर्फ जावास्क्रिप्ट के लिए ही है। वेब पेज़ेस से सम्बंधित जानकारी के लिए Web API और DOM देखें। 

जावास्क्रिप्ट का वर्तमान वर्ज़न ECMAScript 6 या ES 6 कहलाता है। इसे प्रायः ECMAScript 2015 भी कहते हैं।

Javascript  और Java दो अलग प्रोग्रामिंग भाषाएं हैं।
