--- title: Map.prototype.clear() slug: Web/JavaScript/Reference/Global_Objects/Map/clear translation_of: Web/JavaScript/Reference/Global_Objects/Map/clear ---
{{JSRef}}

Method clear() menghapus semua elemen yang ada pada objek Map.

Sintaks

myMap.clear();

Contoh

Menggunakan method clear

var myMap = new Map();
myMap.set("bar", "baz");
myMap.set(1, "foo");

myMap.size;       // 2
myMap.has("bar"); // true

myMap.clear();

myMap.size;       // 0
myMap.has("bar")  // false
