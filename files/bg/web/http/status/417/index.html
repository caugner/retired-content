Кодът за състояние на грешка в HTTP 417 Expectation Failed (Очакванията се провалиха) указва, че очакваното състояние, упоменато като стойност в полето от заглавието на заявката {{HTTPHeader("Expect")}}, не може да бъде удовлетворено.

Вижте {{HTTPHeader("Expect")}} за повече подробности.
