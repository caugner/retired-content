{{DefaultAPISidebar("เซิร์ฟเวอร์ส่งเหตุการณ์")}}

ตามเนื้อผ้าหน้าเว็บต้องส่งคำขอไปยังเซิร์ฟเวอร์เพื่อรับข้อมูลใหม่ นั่นคือหน้าร้องขอข้อมูลจากเซิร์ฟเวอร์ ด้วยเหตุการณ์ที่เซิร์ฟเวอร์ส่งเป็นไปได้ที่เซิร์ฟเวอร์จะส่งข้อมูลใหม่ไปยังหน้าเว็บได้ตลอดเวลาโดยการส่งข้อความไปยังหน้าเว็บ ข้อความขาเข้าเหล่านี้สามารถใช้เป็นข้อมูลกิจกรรม +ในหน้าเว็บ

แนวคิดและการใช้งาน

เพื่อเรียนรู้วิธีการใช้เหตุการณ์เซิร์ฟเวอร์ส่งดูบทความของเราใช้เหตุการณ์เซิร์ฟเวอร์ส่ง

ดูสิ่งนี้ด้วย

เครื่องมือ

  • EventSource polyfill สำหรับ Node.js
  • โพลีฟิลล์EventSourceของชาร์ปของ Remy
  • EventSource Polyfill ของ Yaffle
  • ปลั๊กอิน jqueryของ Rick Waldron
  • intercooler.js สนับสนุน SSE ที่เปิดเผย

หัวข้อที่เกี่ยวข้อง

  • AJAX
  • JavaScript
  • WebSockets

แหล่งข้อมูลอื่น ๆ

  • ทวิตเตอร์เช่นเดียวกับโปรแกรมที่ขับเคลื่อนโดยเหตุการณ์เซิร์ฟเวอร์ที่ส่งและรหัสบน Github
  • เหตุการณ์ HTML5 และเซิร์ฟเวอร์ที่ส่ง
  • เหตุการณ์ที่เซิร์ฟเวอร์ส่งโดยใช้ Asp.Net
