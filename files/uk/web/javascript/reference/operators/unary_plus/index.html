Унарний плюс (+) ставиться перед своїм операндом та повертає значення операнду, але намагається перетворити його на число, якщо він не є числом.

{{EmbedInteractiveExample("pages/js/expressions-unary-plus.html", "taller")}}

Синтаксис

Оператор: +x

Опис

Хоча унарний мінус (-) також вміє перетворювати нечислові значення, унарний плюс найшвидший та найкращий спосіб перетворити щось у число, тому що він не виконує ніяких інших операцій з числом. Він може перетворювати рядкові представлення цілих чисел та чисел з рухомою комою, а також нерядкові значення true, false та null. Підтримуються цілі числа у десятковому та шістнадцятковому (з префіксом 0x) форматах. Підтримуються від'ємні числа (хоча не у шістнадцятковому форматі). Якщо оператор не може розібрати певне значення, він поверне {{jsxref("NaN")}}.

Приклади

Використання з числами

const x = 1;
const y = -1;

console.log(+x);
// 1
console.log(+y);
// -1

Використання з нечисловими значеннями

+true  // 1
+false // 0
+null  // 0
+function(val){ return val } // NaN
