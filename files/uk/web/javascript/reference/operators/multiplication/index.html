Оператор множення (*) повертає добуток операндів.

{{EmbedInteractiveExample("pages/js/expressions-multiplication.html")}}

Синтаксис

Оператор: x * y

Приклади

Множення чисел

 2 * 2      // 4
-2 * 2     // -4

Множення на нескінченність

Infinity * 0         // NaN
Infinity * Infinity  // Infinity

Множення нечислових значень

'foo' * 2 // NaN
