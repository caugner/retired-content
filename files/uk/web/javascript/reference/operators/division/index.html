Оператор ділення (/) повертає частку своїх операндів, де лівий операнд є діленим, а правий операнд є дільником.

{{EmbedInteractiveExample("pages/js/expressions-division.html")}}

Синтаксис

Оператор: x / y

Приклади

Базове ділення

1 / 2              // 0.5

Math.floor(3 / 2) // 1

1.0 / 2.0         // 0.5

Ділення на нуль

2.0 / 0     // Infinity

2.0 / 0.0   // Infinity, тому що 0.0 === 0

2.0 / -0.0  // -Infinity
