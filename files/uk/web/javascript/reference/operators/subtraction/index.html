Оператор віднімання (-) віднімає один операнд від іншого, повертаючи різницю між ними.

{{EmbedInteractiveExample("pages/js/expressions-subtraction.html")}}

Синтаксис

Оператор: x - y

Приклади

Віднімання чисел

5 - 3     // 2
3 - 5     // -2

Віднімання з нечисловими значеннями

'foo' - 3 // NaN
