Властивість __count__ містила кількість перелічуваних властивостей об'єкта, але була прибрана.

Синтаксис

obj.__count__

Приклади

{ 1: 1 }.__count__              // 1
[].__count__                    // 0
[1].__count__                   // 1
[1, /* hole */, 2, 3].__count__ // 3

Специфікації

Не є частиною жодних специфікацій.
