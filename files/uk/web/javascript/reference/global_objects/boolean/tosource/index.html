Метод toSource() повертає строку, що представляє сирцевий код об'єкту.

Синтаксис

booleanObj.toSource()
Boolean.toSource()

Результат

Строка, що представляє сирцевий код об'єкту.

Опис

Метод toSource повертає наступні значення:

  • Для вбудованого об'єкту {{jsxref("Boolean")}}, toSource повертає наступну строку (яка означає, що сирцевий код недоступний): 
    function Boolean() {
    [native code]
}
  • Для екземплярів {{jsxref("Boolean")}}, toSource повертає строку, що представляє сирцевий код об'єкту.

Цей метод зазвичай викликається всередині у JavaScript, а не прямо у коді.

Специфікації

Не є частиною жодного стандарту. Реалізовано у JavaScript 1.3.
