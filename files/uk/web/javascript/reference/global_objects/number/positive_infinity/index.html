Властивість Number.POSITIVE_INFINITY відображає значення позитивної нескінченності.

{{js_property_attributes(0, 0, 0)}}

Опис

Значення Number.POSITIVE_INFINITY є таким самим, як і значення властивості глобального об'єкта {{jsxref("Infinity")}}.

Це значення поводиться дещо інакше, ніж математична нескінченність:

  • Будь-яке позитивне значення, в тому числі POSITIVE_INFINITY, помножене на POSITIVE_INFINITY вертає POSITIVE_INFINITY.
  • Будь-яке від'ємне значення, в тому числі {{jsxref("Number.NEGATIVE_INFINITY", "NEGATIVE_INFINITY")}}, помножене на POSITIVE_INFINITY, вертає {{jsxref("Number.NEGATIVE_INFINITY", "NEGATIVE_INFINITY")}}.
  • Будь-яке позитивне число, розділене на POSITIVE_INFINITY, вертає нуль.
  • Будь-яке від'ємне число, розділене на POSITIVE_INFINITY, вертає від'ємний нуль.
  • Нуль, помножений на POSITIVE_INFINITY, вертає {{jsxref("NaN")}}.
  • Значення {{jsxref("Global_Objects/NaN", "NaN")}}, помножене на POSITIVE_INFINITY, вертає {{jsxref("NaN")}}.
  • Значення POSITIVE_INFINITY, розділене на будь-яке від'ємне значення, окрім {{jsxref("Number.NEGATIVE_INFINITY", "NEGATIVE_INFINITY")}}, вертає {{jsxref("Number.NEGATIVE_INFINITY", "NEGATIVE_INFINITY")}}.
  • Значення POSITIVE_INFINITY, розділене на будь-яке позитивне значення, окрім POSITIVE_INFINITY, вертає POSITIVE_INFINITY.
  • Значення POSITIVE_INFINITY, розділене на {{jsxref("Number.NEGATIVE_INFINITY", "NEGATIVE_INFINITY")}} чи POSITIVE_INFINITY, вертає {{jsxref("NaN")}}.

Ви можете скористатись властивістю Number.POSITIVE_INFINITY, щоб позначити помилку в умові, яка вертає скінченне число у випадку успіху. Зауважте, однак, що метод {{jsxref("isFinite")}} в цьому випадку більш доречний.

Оскільки POSITIVE_INFINITY є статичною властивістю об'єкта {{jsxref("Число", "Number")}}, її завжди слід використовувати як Number.POSITIVE_INFINITY, а не як властивість створеного вами об'єкта {{jsxref("Число", "Number")}}. 

Приклади

Використання POSITIVE_INFINITY

У наступному прикладі змінній bigNumber присвоюється значення, яке є більшим за максимальне. Коли виконується інструкція {{jsxref("Statements/if...else", "if")}}, bigNumber має значення Infinity, а отже, bigNumber присвоюється більш кероване значення для продовження.

var bigNumber = Number.MAX_VALUE * 2;

if (bigNumber == Number.POSITIVE_INFINITY) {
  bigNumber = returnFinite();
}
