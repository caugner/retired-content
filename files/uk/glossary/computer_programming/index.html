Комп'ютерне програмування - це процес складання та організації колекції інструкцій. Вони кажуть комп'ютеру/програмному забезпеченню, що робити, тією мовою, яку розуміє комп'ютер. Ці інструкції подаються у формі багатьох різних мов, таких як C++, Java, JavaScript, HTML, Python, Ruby та Rust.  

Використовуючи відповідну мову, ви можете запрограмувати/створити найрізноманітніші види програмного забезпечення. Наприклад, програму, яка допомагає вченим зі складними розрахунками, базу даних, що зберігає величезні обсяги даних, вебсайт, який дозволяє завантажувати музику, або програмне забезпечення для анімації, яке дозволяє створювати анімаційні фільми.
