Інтегроване Середовище Розробки (IDE) або Інтерактивне Середовище Розробки це програмне забезпечення що надає комплексні можливості для програмістів для розробки програмного забезпечення. IDE зазвичай складається з редактора вихідного коду, засобів автоматизації побудови та відладчика.
