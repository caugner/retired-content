Об'єкт являє собою структуру, що містить власне дані та інструкції для обробки даних. Часом об'єкти стосуються справжніх речей, як-от авто чи мапа у грі в автомобільні перегони. Прикладами {{glossary("OOP", "об'єктно-орієнтованих мов програмування")}} є {{glossary("JavaScript")}}, Java, C++, Python, Ruby тощо.
