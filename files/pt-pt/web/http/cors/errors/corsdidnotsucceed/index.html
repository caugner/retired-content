Motivo

Motivo: O pedido CORS não foi concluído com sucesso

O que correu mal?

O {{Glossary("HTTP")}} pedido que faz uso do CORS falhou porque a conexão HTTP falhou a nivel da rede ou do protocolo de conexão. O error não está directamente relacionado com CORS, mas é algum tipo de erro da rede.
