O código de resposta HTTP (Protocolo de Transferência de Hipertexto) 502 Bad Gateway indica que o servidor, ao agir como portal ou proxy, recebeu uma resposta invalida dum servidor upstream (um servidor mais adiante na cadeia de conexões).

Nota: Uma {{interwiki("wikipedia", "Gateway", "Gateway")}} pode ser várias coisas numa rede e um erro 502 tipicamente não pode ser resolvido por si, mas requer um conserto no servidor ou nas proxies por qual esta a comunicar.

502 Bad Gateway
