O código de resposta HTTP (Protocolo de Transferência de Hipertexto) 504 Gateway Timeout indica que o servidor, ao agir como portal ou proxy, não recebeu uma resposta dum servidor upstream (um servidor mais adiante na cadeia de conexões) no tempo necessário para completar a resposta. 

Nota: Uma {{interwiki("wikipedia", "Gateway", "Gateway")}} pode ser várias coisas numa rede e um erro 504 tipicamente não pode ser resolvido por si, mas requer um conserto no servidor ou nas proxies por qual esta a comunicar.

Status

504 Gateway Timeout
