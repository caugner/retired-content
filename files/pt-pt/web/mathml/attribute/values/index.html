Comprimentos

Vários elementos de apresentação MathML têm atributos que aceitam valores de comprimento usados para dimensões ou espaçamento. MathML aceita diferentes unidades e constantes para indicar comprimentos.

Unidades

Unidade Descrição
em Unidade {{ Cssxref("font-size", "relativa à fonte") }}
ex Unidade {{ Cssxref("font-size", "relativa à fonte") }}. (A altura "x" do elemento, 1ex ≈ 0.5em em muitas fontes)
px Pixeis
in Polegadas (1 polegada = 2,54 centímetros)
cm Centímetros
mm Milímetros
pt Pontos (1 ponto = 1/72 polegada)
pc Picas (1 pica = 12 pontos)
% Percentagem do valor padrão.
