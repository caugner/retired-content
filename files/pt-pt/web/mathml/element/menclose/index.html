{{MathMLRef}}

O elemento de MathML <menclose> apresenta os seus conteúdos dentro da notação indicada pelo atributo notation.

Atributos

class, id, style
Para uso com stylesheets.
href
Usado para criar uma hiperligação para um URI.
mathbackground
A cor de fundo. Pode usar #rgb, #rrggbb e nomes de cores HTML.
mathcolor
A cor do texto. Pode usar #rgb, #rrggbb e nomes de cores HTML.
notation
A lista de notações, separada por espaços, para aplicar aos elementos filhos. Os símbolos são desenhados como se os outros não estivessem presentes, e por isso podem sobrepor-se. Os valores possíveis são:
Valor Exemplo de apresentação Apresentação no seu navegador Descrição
longdiv (default) a2 + b2 símbolo de longa divisão
actuarial a2 + b2 símbolo atuarial
radical {{deprecated_inline}} a2 + b2 símbolo de raiz quadrada. Depreciado. Use {{MathMLElement("msqrt")}}.
box a2 + b2 caixa
roundedbox a2 + b2 caixa arredondada
circle a2 + b2 circulo
left a2 + b2 linha vertical à esquerda dos conteúdos
right a2 + b2 linha vertical à direita dos conteúdos
top a2 + b2 linha horizontal acima dos conteúdos
bottom a2 + b2 linha horizontal abaixo dos conteúdos
updiagonalstrike a2 + b2 linha através do conteúdo da parte inferior esquerda para a parte superior direita
downdiagonalstrike a2 + b2 linha através do conteúdo da parte superior esquerda para a parte inferior direita
verticalstrike a2 + b2 linha vertical através do conteúdo
horizontalstrike a2 + b2 linha horizontal através do conteúdo
madruwb a2 + b2 símbolo fatorial arábico
updiagonalarrow a2 + b2 seta diagonal
phasorangle a2 + b2 ângulo fasor

Exemplos

<math>

  <menclose notation="circle box">
    <mi> x </mi>
    <mo> + </mo>
    <mi> y </mi>
  </menclose>

</math>
