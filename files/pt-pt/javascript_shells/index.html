Um shell JavaScript permite a você rapidamente testar partes de código JavaScript sem ter que atualizar a página web. Eles são extremamente úteis para desenvolvimento e depuração de código.

Lista de shells JavaScript

Os seguintes shells JavaScript trabalham com o Mozilla:

  • JavaScript Shell (página web) - também disponível como parte da extensão Developer's Extension
  • JavaScript Shell (js) - Um intérprete de linha de comando para JavaScript (requere sua própria construção Mozilla).
    • xpcshell é um XPConnect- shell habilitado, algumas vezes útil para desenvolvimento Mozilla.
  • JavaScript Shell Server - permite conexões de Emacs.
  • rainbow9 - Javascript IDE
